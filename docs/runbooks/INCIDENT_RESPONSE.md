# Incident Response Runbook (Stub)

**Owner:** Security Lead  
**Activation:** Sprint 14 security prep

## Severity levels

| Level | Description                      | Response time |
| ----- | -------------------------------- | ------------- |
| P0    | Active breach, data exfiltration | 15 min        |
| P1    | Secret leak, tenant crossover    | 1 hour        |
| P2    | Vulnerability discovered         | 24 hours      |

## Immediate actions (P0/P1)

1. Page on-call engineer
2. Isolate affected service
3. Rotate compromised credentials
4. Preserve logs with correlation IDs
